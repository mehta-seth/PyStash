Perfect Number

Perfect number -number whose divisors (except itself) add up to itself.

For example, 28 is a perfect number.
Its proper divisors are {1, 2, 4, 7, 14} and 1 +2 +4 +7 +14 = 28.

Code:


def perfect_no(n):
    sum=0
    for i in range(1,n):
        if n%i==0:
            sum=sum+i
    return sum
def check(n):
    if n==perfect_no(n):
        print(n," IS PERFECT NUMBER")
    else:
        print(n," IS NOT A PERFECT NUMBER")
n=int(input("ENTER A NUMBER TO CHECK IF IT IS PERFECT NUMBER OR NOT"))
check(n)

Output:

ENTER A NUMBER TO CHECK IF IT IS PERFECT NUMBER OR NOT 28

28 IS PERFECT NUMBER
